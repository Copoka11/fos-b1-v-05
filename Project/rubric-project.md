# Рубрика проектной работы

Максимум за проект — **25 баллов**.

| Критерий | Баллы |
|---|---:|
| Постановка задачи, метрика и baseline | 0–3 |
| Данные, разбиение и отсутствие утечки | 0–4 |
| Корректность реализации и воспроизводимость | 0–4 |
| Экспериментальный дизайн и сравнение моделей | 0–4 |
| Качество результата и анализ ошибок | 0–3 |
| Документация и инструкция запуска | 0–3 |
| Обоснование решений и представление результатов | 0–2 |
| Ограничения, этика, источники и использование ИИ | 0–2 |
| **Итого** | **25** |

Критический дефект должен быть устранён независимо от промежуточной суммы баллов.
