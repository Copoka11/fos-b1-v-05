# Модуль 2. Нейросетевые методы классификации

Модуль формирует результаты DL-1.1–DL-1.4 на уровне С.

| КИМ | Назначение | Индикаторы | Уровень | Баллы |
|---|---|---|:---:|---:|
| [ЛР5](kim-05-lab-5.md) | MLP для табличных данных | DL-1.1, DL-1.2 | С | 8 |
| [ЛР6](kim-06-lab-6.md) | CNN и transfer learning | DL-1.3, DL-1.4 | С | 8 |

Все лабораторные оцениваются по [единой рубрике 0–8](../methodical-guidelines/teachers-assessment/rubric-laboratory.md). Вопросы по нейронным сетям входят в [единый текущий и рубежный контроль](../M1-classical-ml/kim-05-current-and-milestone-control.md).
