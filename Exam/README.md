# Зачёт и защита проекта

Промежуточная аттестация проводится в форме защиты сквозного проекта и ответов на вопросы. Она подтверждает освоение дисциплины в целом и не дублирует отдельную лабораторную работу.

## Сценарий

1. Доклад — 7 минут.
2. Демонстрация воспроизводимого запуска — 3 минуты.
3. Ответы на вопросы комиссии — 5 минут.

## Оценивание

Максимум за защиту — **10 баллов**, минимум для зачёта по защите — **4 балла**. Общий результат по дисциплине должен составлять не менее **60/100**.
