# Зачёт и защита проекта

Промежуточная аттестация проводится в форме защиты сквозного проекта и ответов на вопросы. Она подтверждает освоение дисциплины в целом и не дублирует отдельную лабораторную работу.

- [критерии комиссии](../docs/commission_criteria.md);
- [шкала защиты](../src/rubrics/commission_scale.md);
- [форма комиссии](../src/templates/commission_form.md);
- [чек-лист перед защитой](../release/printable/checklist_before_defense.md).

Максимум за защиту — **10 баллов**. Минимум для зачёта по защите — **4 балла**, общий результат по дисциплине — не менее **60/100**.
