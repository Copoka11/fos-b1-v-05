# Чек-лист перед защитой

- [ ] Данные доступны либо приведена инструкция их получения.
- [ ] Проект запускается по инструкции.
- [ ] Результаты воспроизводятся.
- [ ] Исключена утечка тестовых данных и целевой переменной.
- [ ] Peer-review проведён, замечания обработаны.
- [ ] Презентация содержит обязательные элементы.
- [ ] Зафиксирован вклад каждого участника.
- [ ] Указаны роли КРМ, подтверждаемые проектом.
