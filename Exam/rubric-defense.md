# Рубрика защиты

Максимум — **10 баллов**.

| Критерий | Баллы |
|---|---:|
| Постановка задачи, данные и метрика | 0–2 |
| Методическая корректность эксперимента | 0–2 |
| Достоверность результатов и анализ ошибок | 0–2 |
| Демонстрация и воспроизводимость | 0–1 |
| Обоснование решений и ограничений | 0–1 |
| Ответы на вопросы комиссии | 0–2 |
| **Итого** | **10** |

Интерпретация: 9–10 — самостоятельно и доказательно; 7–8 — уверенно с небольшими пробелами; 5–6 — основное подтверждено; 4 — минимальный порог защиты; 0–3 — результат не подтверждён.
