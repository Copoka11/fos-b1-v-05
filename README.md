# ФОС Б1.В.05 «Методы искусственного интеллекта в задачах классификации»

Комплексный фонд оценочных средств проектной работы для направления 02.03.03 «Математическое обеспечение и администрирование информационных систем», профиль «Искусственный интеллект и аналитика данных», 3 курс, 6 семестр, 2 з.е. (72 ч.), форма аттестации — зачёт.

## Быстрый старт
1. Преподаватель публикует `FOS_BRIEF.md`, `docs/project_criteria.md` и `src/templates/student_selfcheck.md`.
2. Студент выбирает задачу классификации и формирует воспроизводимый репозиторий проекта.
3. Оценивание проводится по лабораторным работам, проекту, peer-review и защите.
4. Баллы суммируются до 100; зачёт — от 60 баллов при выполнении обязательных условий.
