## 1. О дисциплине

| Параметр | Значение |
|---|---|
| Код и название | Б1.В.05 «Методы искусственного интеллекта в задачах классификации» |
| Направление | 02.03.03 «Математическое обеспечение и администрирование информационных систем» |
| Профиль | «Искусственный интеллект и аналитика данных» |
| Курс / семестр | 3 курс / 6 семестр |
| Объём | 2 зачётные единицы / 72 часа |
| Форма аттестации | зачёт в форме защиты проекта и ответов на вопросы |

**Цель дисциплины** — сформировать знания и практические навыки выбора, реализации, оценивания и интерпретации классических и нейросетевых моделей классификации с учётом свойств данных, ограничений методов и требований к воспроизводимости.

Дисциплина включает два модуля, шесть лабораторных работ, текущий и рубежный контроль, сквозной проект, peer-review и защиту. Входная диагностика проводится без начисления баллов.

## 4. Итоговая оценка

| Оценочное средство | Баллы |
|---|---:|
| ЛР1–ЛР6, по 8 баллов | 48 |
| Текущий и рубежный контроль | 12 |
| Сквозной проект | 25 |
| Peer-review | 5 |
| Защита | 10 |
| **Итого** | **100** |

Для зачёта необходимо набрать не менее 60 баллов, включая не менее 15/25 за проект и не менее 4/10 за защиту, а также устранить критические дефекты.
