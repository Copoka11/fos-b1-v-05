# ФОС Б1.В.05 «Методы искусственного интеллекта в задачах классификации»

Комплексный фонд оценочных средств проектной работы для направления 02.03.03 «Математическое обеспечение и администрирование информационных систем», профиль «Искусственный интеллект и аналитика данных», 3 курс, 6 семестр, 2 з.е. (72 ч.), форма аттестации — зачёт.

Структура публикации согласована с репозиторием-шаблоном [`Vl-Tershch/fos-repository-template`](https://github.com/Vl-Tershch/fos-repository-template), при этом сохранены расширенные материалы, предусмотренные ТЗ проекта: ADR, рубрики, формы, валидация и печатные версии.

## Быстрый старт
1. Преподаватель публикует `FOS_BRIEF.md`, `docs/project_criteria.md` и `src/templates/student_selfcheck.md`.
2. Студент выбирает задачу классификации и формирует воспроизводимый репозиторий проекта.
3. Оценивание проводится по лабораторным работам, проекту, peer-review и защите.
4. Баллы суммируются до 100: лабораторные работы — 48, текущий и рубежный контроль — 12, проект — 25, peer-review — 5, защита — 10; зачёт — от 60 баллов при выполнении обязательных условий.

## Модель измерения

| Элемент | Индикаторы | Уровень | Основные КИМ |
|---|---|:---:|---|
| Постановка и данные | ML-2.1, ML-2.2 | Б | ЛР1, текущий/рубежный контроль, проект |
| Метрики и дисбаланс | ML-2.3 | С | ЛР2, текущий/рубежный контроль, проект, peer-review |
| Классические модели | ML-3.1, ML-3.2, ML-3.3 | С | ЛР3–ЛР4, текущий/рубежный контроль, проект, защита |
| Обучение MLP | DL-1.1, DL-1.2 | С | ЛР5, текущий/рубежный контроль, проект, защита |
| CNN и глубокие архитектуры | DL-1.3, DL-1.4 | С | ЛР6, текущий/рубежный контроль, проект |

Развёрнутая таблица с предметными дескрипторами и ресурсами находится в [`docs/measurement_model.md`](docs/measurement_model.md).
