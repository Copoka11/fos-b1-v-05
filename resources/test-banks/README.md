# Банк тестов и контрольных вопросов

Основной банк вопросов, мини-заданий и рубежных заданий размещён в КИМ [«Текущий и рубежный контроль»](../../M1-classical-ml/kim-05-current-and-milestone-control.md).

При формировании вариантов преподаватель сохраняет проверяемый индикатор, уровень Б/С, сопоставимую сложность и критерии 0–2 или 0–3 балла.
