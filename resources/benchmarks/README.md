# Baseline и бенчмарки

## Обязательные baseline

1. `DummyClassifier` со стратегией, соответствующей дисбалансу классов.
2. Логистическая регрессия с корректным preprocessing.
3. Для изображений — простая CNN либо линейный классификатор по фиксированным признакам.

## Сильные ориентиры

- табличные данные: Random Forest и один градиентный boosting;
- изображения: компактная CNN и transfer learning с предобученной архитектурой;
- конкретный SOTA не является обязательным: он быстро меняется и может требовать несопоставимых ресурсов.

## Протокол сравнения

Все модели сравниваются на одинаковых разбиениях и по заранее выбранным метрикам. Таблица должна включать:

| Модель | Основная метрика | Дополнительная метрика | Время обучения | Время инференса | Число параметров/размер | Ограничения |
|---|---:|---:|---:|---:|---:|---|

Для CV приводятся среднее, стандартное отклонение и число фолдов. Test используется однократно для итоговой оценки.
