# Научные статьи

Для проектной работы допускаются статьи из рецензируемых журналов и конференций по выбранной задаче классификации. В карточке статьи указываются полная ссылка, решаемая задача, данные, метрики, ключевой результат, ограничения и связь с КИМ.

Конкретный список статей зависит от выбранного домена и актуализируется перед началом потока.
