# Аннотированная литература

1. **C. Bishop. Pattern Recognition and Machine Learning.** Теоретическая основа вероятностной классификации, линейных моделей, ядерных методов и оценки качества. Используется при обосновании ML-3.1 и ML-3.3.
2. **T. Hastie, R. Tibshirani, J. Friedman. The Elements of Statistical Learning.** Источник для сопоставления линейных моделей, SVM, деревьев и ансамблей. Полезен при подготовке рубежного сравнения моделей.
3. **I. Goodfellow, Y. Bengio, A. Courville. Deep Learning.** Основной источник по функциям потерь, оптимизации, MLP и CNN; поддерживает DL-1.1—DL-1.4.
4. **A. Géron. Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow.** Практический справочник по воспроизводимым pipeline и реализации лабораторных работ.
5. **Документация scikit-learn:** <https://scikit-learn.org/stable/user_guide.html>. Первоисточник по API, preprocessing, model selection и метрикам. Следует использовать вместо случайных примеров из блогов.
6. **Документация PyTorch:** <https://pytorch.org/docs/stable/index.html>. Первоисточник по слоям, функциям потерь, оптимизаторам и воспроизводимости нейросетевого эксперимента.

При использовании конкретной главы студент указывает раздел и объясняет, какое решение проекта принято на её основе.
