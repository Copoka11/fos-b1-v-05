# Информационные ресурсы

Ресурсы сгруппированы по типовой структуре шаблона. В каждом разделе указаны назначение и связь с КИМ.

| Тип | Раздел | Назначение |
|---|---|---|
| Учебники | [textbooks](textbooks/README.md) | базовые и дополнительные учебные материалы |
| Научные статьи | [papers](papers/README.md) | рекомендации по подбору современных исследований |
| Банки тестов | [test-banks](test-banks/README.md) | текущий и рубежный контроль |
| Банки задач | [problem-banks](problem-banks/README.md) | структура лабораторного и проектного репозитория |
| Датасеты | [datasets](datasets/README.md) | учебные и проектные данные |
| Бенчмарки | [benchmarks](benchmarks/README.md) | baseline и протокол сравнения |
| Python-библиотеки | [software/python-libs](software/python-libs/README.md) | инструменты реализации и воспроизводимости |
| LLM и промпты | [llm-prompts](llm-prompts/README.md) | проверка решений и раскрытие использования ИИ |
| Прочее | [other](other/README.md) | дополнительные ресурсы |

Использование ресурсов не заменяет самостоятельного обоснования. Студент указывает версии библиотек, источники данных и факт применения генеративного ИИ.
