# Данные КРМ

| Файл или каталог | Назначение |
|---|---|
| Исходная книга КРМ 3.0 | используется для сверки; не публикуется в открытом репозитории |
| [`competency_map/competencies.yaml`](competency_map/competencies.yaml) | выбранные компетенции, индикаторы и дескрипторы |
| [`competency_map/roles.yaml`](competency_map/roles.yaml) | целевые профессиональные роли |
| [`competency_map/mapping.csv`](competency_map/mapping.csv) | машиночитаемая связь индикаторов и оценочных средств |

Исходная книга хранится у разработчика и не редактируется. В репозитории размещается только отобранное машиночитаемое соответствие: исходные коды и формулировки КРМ, предметные дескрипторы и целевые уровни Б/С.
