# Данные КРМ

| Файл или каталог | Назначение |
|---|---|
| `competency_map/competencies.yaml` | выбранные компетенции, индикаторы, уровни и дескрипторы |
| `competency_map/roles.yaml` | целевые профессиональные роли |
| `competency_map/mapping.csv` | машиночитаемая связь индикаторов и оценочных средств |

Исходная книга КРМ 3.0 используется для сверки, но не публикуется в репозитории дисциплины. В открытой части размещается отобранное соответствие, необходимое для проверки ФОС. Исходные коды и формулировки сохраняются, предметные дескрипторы и целевые уровни Б/С указываются отдельно.
