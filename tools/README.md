# Инструменты проверки

Скрипт [`validate_repository.py`](validate_repository.py) запускается стандартным Python без сторонних зависимостей:

```bash
python tools/validate_repository.py
```

Проверяются обязательные файлы, внутренние ссылки, структура баллов, лабораторные КИМ, индикаторы КРМ, минимальное покрытие и состав команды.
