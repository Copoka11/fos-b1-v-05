# Состав фонда оценочных средств

| Форма контроля | Назначение | КИМ | Результат |
|---|---|---|---|
| Входная диагностика | Проверка пререквизитов | [Диагностические задания](../src/kims/current_control.md#входная-диагностика) | маршрут повторения, без баллов |
| Текущий контроль | Проверка каждой темы | [Вопросы и мини-задания](../src/kims/current_control.md) | обратная связь на занятии |
| Лабораторные работы | Практическое подтверждение индикаторов | [Комплект ЛР](../src/kims/laboratory_works.md) | код, отчёт, метрики |
| Рубежный контроль | Проверка модулей и готовности проекта | [Рубежные задания](../src/kims/milestone_control.md) | решение и устное объяснение |
| Проектная работа | Комплексная междисциплинарная проверка | [Постановка проекта](../FOS_BRIEF.md) | репозиторий, отчёт, презентация |
| Peer-review | Независимая проверка решения | [Форма рецензии](../src/templates/peer_review_form.md) | рецензия и ответ автора |
| Промежуточная аттестация | Подтверждение итогового результата | [Зачёт и защита](../src/kims/final_assessment.md) | итоговая оценка |

Критерии, веса и пороги приведены в [паспорте ФОС](passport.md) и [модели измерения](measurement_model.md).
