# Пререквизиты и междисциплинарные связи по КРМ

## Входные результаты

| Предшествующая подготовка | Ожидаемый входной результат | Связь с КРМ | Входной уровень |
|---|---|---|:---:|
| Программирование на Python | Пишет функции, работает с массивами, файлами и библиотеками | база для ML-2.1 и ML-2.2 | Б |
| Обработка данных на Python | Загружает, очищает, преобразует и визуализирует данные | входной ML-2.2 | Б |
| Теория вероятностей и математическая статистика | Понимает выборку, распределение, оценку и статистическую погрешность | основа ML-2.3 | Б |
| Машинное обучение | Различает обучение с учителем и без учителя, понимает train/test split | входной ML-2.1 | Б |
| Математические модели нейронных сетей | Понимает нейрон, функцию активации, функцию потерь и градиентный спуск | входной DL-1.1 | Б |

## Диагностика входа

На первом занятии проводится диагностический тест и короткое практическое задание: загрузить набор данных, выделить признаки и целевую переменную, выполнить стратифицированное разбиение и объяснить назначение тестовой выборки. Диагностика не входит в итоговые 100 баллов.

## Выходные связи

Результаты дисциплины используются при выполнении НИРС, практики, выпускной квалификационной работы и дисциплин по глубокому обучению, компьютерному зрению, анализу текстов и промышленной разработке систем ИИ.

| Последующая деятельность | Используемые результаты |
|---|---|
| НИРС | ML-2.3, ML-3.3, DL-1.1: гипотеза, эксперимент, анализ ошибок |
| Производственная практика | ML-2.2, ML-3.2: подготовка данных и реализация модели |
| ВКР | ML-2.1—2.3, ML-3.1—3.3, DL-1.1—1.4: полный цикл исследования |
