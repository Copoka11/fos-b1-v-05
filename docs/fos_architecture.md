# Архитектура ФОС

- `meta` фиксирует входные данные, границы и архитектурные решения;
- `docs` содержит целевую РПД, структуру курса, паспорт и модель измерения;
- `data/competency_map` хранит машиночитаемое соответствие ролей, индикаторов и средств;
- `src/kims` содержит контрольно-измерительные материалы;
- `src/rubrics` и `src/templates` содержат критерии и формы оценивания;
- `resources` каталогизирует литературу, данные, инструменты, baseline, шаблоны и промпты;
- `methodology` содержит указания для студентов и преподавателей;
- `tests` автоматически проверяет целостность и внутренние ссылки;
- `validation` используется для калибровки рубрик;
- `release` обеспечивает внедрение и печатные материалы.

Центральная навигация между элементами реализована в [модели измерения компетенций](measurement_model.md).
