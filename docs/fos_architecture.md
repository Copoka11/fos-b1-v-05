# Архитектура ФОС
`meta` фиксирует решения и входы; `docs` содержит нормативную логику; `data/competency_map` — машиночитаемое соответствие; `src/rubrics` и `src/templates` — исполняемые формы оценивания; `tests` проверяет целостность; `validation` калибрует рубрики; `release` обеспечивает внедрение.
