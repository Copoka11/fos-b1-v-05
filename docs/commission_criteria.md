# Защита перед комиссией
Доклад 7 минут, демонстрация 3 минуты, вопросы 5 минут. Обязательны: задача и данные; схема эксперимента; baseline; сравнение моделей; анализ ошибок; ограничения; вклад участников. Критерии 0–10: ясность постановки, методическая корректность, обоснование решений, достоверность результатов, демонстрация, ответы на вопросы, ролевой вклад по КРМ.
