# Паспорт ФОС
## Назначение
Единообразная проверка достижения результатов по ML-2, ML-3 и DL-1 в ходе лабораторных работ и сквозного проекта классификации.

## Средства и веса
| Средство | Вес |
|---|---:|
| ЛР1: EDA, разбиение данных, базовый pipeline | 8% |
| ЛР2: Метрики, кросс-валидация и дисбаланс классов | 8% |
| ЛР3: Логистическая регрессия и SVM | 8% |
| ЛР4: Деревья решений, Random Forest и boosting | 8% |
| ЛР5: MLP для классификации табличных данных | 8% |
| ЛР6: CNN и transfer learning для изображений | 8% |
| Текущий и рубежный контроль | 12% |
| Проект: Сквозное программное решение задачи классификации | 25% |
| Peer-review: Перекрестная экспертиза проекта | 5% |
| Защита: Презентация и ответы комиссии | 10% |

В составе единого средства «Текущий и рубежный контроль» до 6 баллов начисляется за три тематические текущие проверки и до 6 баллов — за два рубежных контроля. Входная диагностика проводится без начисления баллов.

## Условия зачёта
- итог не менее 60/100;
- проект сдан и получил не менее 15/25;
- защита не ниже 4/10;
- нет критического дефекта: утечки целевой переменной/тестовой выборки, невоспроизводимого результата или заимствования без атрибуции.

## Шкала
90–100 — высокий уровень; 75–89 — повышенный; 60–74 — пороговый; 0–59 — результаты не подтверждены. Для отдельных рубрик используется шкала 0–10 с опорными уровнями 10/8/6/4/0–3.
