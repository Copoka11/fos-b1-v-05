# Методические указания преподавателям по использованию ресурсов

Правила подбора ресурсов и автоматизированной проверки КИМ приведены в [`methodology/resource_usage.md`](../../methodology/resource_usage.md).

При обновлении КИМ необходимо проверить источник данных, лицензию, работоспособность ссылки, актуальность API, соответствие индикатору и отсутствие раскрытия готового ответа.
