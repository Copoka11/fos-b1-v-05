# Методические указания преподавателям по использованию ресурсов

## Подбор ресурсов

Для каждого источника проверяются:

- официальный источник и авторство;
- лицензия и условия доступа;
- дата проверки и актуальность API;
- связь с конкретным индикатором и КИМ;
- отсутствие раскрытия готового ответа.

## Формирование задания

Задание строится по схеме: контекст → входные данные → требуемый артефакт → ограничения → критерии → проверяемое свидетельство. Набор данных и его версия фиксируются до начала оценивания.

## Генеративный ИИ

Разрешается применять [каталог промптов](../../resources/llm-prompts/README.md) для первичного анализа и рецензирования. Преподаватель проверяет фактическую корректность, уровень сложности, соответствие индикатору и конфиденциальность данных.

## Автоматическая проверка

Команда `python tools/validate_repository.py` проверяет структуру, ссылки, баллы, коды индикаторов и минимальное покрытие.
