# Методические указания преподавателям по оцениванию

Порядок проверки, обратной связи и применения рубрик приведён в [методических указаниях преподавателю](../../methodology/teachers.md).

Для лабораторных используется [единая шкала 0–8](../../src/rubrics/laboratory_scale.md). Текущие и рубежные проверки образуют одно средство весом 12 баллов и не дублируют баллы лабораторных работ.
