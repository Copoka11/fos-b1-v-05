# Использование ресурсов при создании КИМ

## Ручной режим

Преподаватель выбирает набор данных, фиксирует его версию, определяет целевой индикатор и формирует задание по шаблону: контекст → входные данные → требуемый артефакт → ограничения → критерии → проверяемое свидетельство.

## Автоматизированный режим

Скрипт [`tests/validate_fos.py`](../tests/validate_fos.py) проверяет структуру ФОС, сумму уникальных весов, известность индикаторов и минимальное покрытие. Код выполняется стандартным Python без дополнительных библиотек.

## Использование БЯМ

Разрешается применять [каталог промптов](../resources/prompts.md) для генерации вариантов вопросов и первичного рецензирования. Преподаватель обязан проверить фактическую корректность, соответствие индикатору, уровень сложности и отсутствие раскрытия ответа в условии.
