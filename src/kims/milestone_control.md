# Рубежный контроль

## Рубеж 1. Данные и оценивание

Студент получает новый набор данных и за 45 минут должен:

1. сформулировать задачу классификации;
2. предложить схему разбиения без утечки;
3. выбрать baseline и две метрики;
4. назвать риски качества данных;
5. письменно обосновать решения.

## Рубеж 2. Сравнение моделей

По результатам ЛР1—ЛР4 студент представляет таблицу минимум из четырёх моделей, сопоставляя качество, время обучения, время инференса, объяснимость и ограничения. Затем отвечает на два вопроса преподавателя.

## Критерии

Каждый рубеж оценивается максимум в 3 балла.

| Критерий | Баллы за рубеж |
|---|---:|
| Методическая корректность решения, включая отсутствие утечки | 0–1 |
| Обоснованность метрик, схемы валидации и сопоставимость свидетельств | 0–1 |
| Анализ ограничений и самостоятельность объяснения | 0–1 |
| **Итого за один рубеж** | **0–3** |

Два рубежных контроля дают максимум 6 баллов. Вместе с [текущими проверками](current_control.md), также оцениваемыми максимум в 6 баллов, они образуют единое оценочное средство «Текущий и рубежный контроль» весом 12 баллов.
